* Hisilicon HiX5HD2 MDIO Ethernet Controller interface

Required properties:
- compatible: should be "hisilicon,higmac-mdio".
- reg: address and length of the register set for the device.

Example at the SoC level:
mdio@f89403c0 {
	compatible = "hisilicon,higmac-mdio";
	reg = <0xf89403c0 0x14>;
	#address-cells = <1>;
	#size-cells = <0>;
};

And at the board level:

mdio@f89403c0 {
	phy0: ethernet-phy@0 {
		reg = <0>;
	};
};
